Class BoundingBox with Generic (T) is for times when you have points you want to save on the boundary of the bounding box. these points are of type T, which is constrained to be an IVertex3D - currently instantiated by Vertex and Vector3 Implements the BoundingBox

Properties

PointsOnFaces

The PointsOnFaces is an array of 6 lists which are vertices of the tessellated solid that are on the faces of the bounding box. They are in the order of direction1-low, direction1-high, direction2-low, direction2-high, direction3-low, direction3-high.
